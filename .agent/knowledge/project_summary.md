# Navy Archive: Ocean Edition - Project Knowledge Base

## 1. Branding & Identity
- **Project Name**: Navy Archive (해군 디지털 피복) - Ocean Edition
- **Concept**: Commercial Tactical Luxe (FashionPlus Inspired)
- **Core Theme**: High-density professional fashion portal for ROK Navy personnel.
- **Color Palette**:
    - **Navy (Primary)**: `#1e3b5a` (Deep Maritime)
    - **Red (Accent)**: `#EE1C23` (Tactical Point)
    - **White (Base)**: `#FFFFFF` (Clean Commercial)
- **Typography**:
    - **Noto Sans KR**: UI & Navigation (Professional)
    - **S-Core Dream**: Content & Descriptions (Tech/Modern)
    - **Bold Italic**: Headline Impact (Dynamic)

## 2. Key Feature Modules
- **Landing Page**: High-impact visual storytelling with red/navy gradients and verified maritime imagery.
- **Premium Mall**: FashionPlus-style high-density commerce portal.
    - Utility Top-bar + Centered Search + Category GNB.
    - Multi-banner Hero sections.
    - Specialized Product Grid (4-5 items per row, asymmetric crops).
- **Tip & Knowledge**: Post-style informational module for shipboard life.
- **Responsive Layout**: Compact, no-scroll dashboard style (1440px max-width).
- **Accessibility**: ARIA-compliant Dialog titles, keyboard-navigable components.

## 3. Page Structure
- `/`: Main Landing Page (Dashboard style)
- `/tips`: Shipboard Life Tips (함상 생활 가이드) - Informational resources for personnel.

## 4. Maintenance / AI Guidance
- **Design Rule**: Always maintain the "Modern Tactical Luxe" aesthetic. Avoid cluttered layouts.
- **Iconography**: Use `lucide-react`. Maintain consistency with Anchor, Waves, and Shield icons.
- **Localization**: Use formal, polished Korean terminology suitable for ROK Navy personnel.
