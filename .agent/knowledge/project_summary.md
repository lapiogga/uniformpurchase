# Navy Archive: Ocean Edition - Project Knowledge Base

## 1. Branding & Identity
- **Project Name**: Navy Archive (해군 디지털 피복)
- **Concept**: Modern Tactical Luxe (Uniqlo-inspired Minimalism)
- **Core Theme**: Republic of Korea Navy (ROK Navy) "Deep Blue Honor"
- **Color Palette**: 
  - **Uniqlo Red**: `#EE1C23` (Action points, Branding)
  - **Navy Blue**: Deep blue tones for filters and overlays
  - **Black/White**: Clean, high-contrast base
- **Typography**: 
  - **Headings**: Noto Sans KR (본고딕) - Bold/Black weights
  - **Body**: S-Core Dream (에스코어 드림) - Light/Medium weights

## 2. Technical Key Features
- **Hero Slideshow**: 6-image cyclical rotation with 3-second intervals, fade-in transitions, and z-index management.
- **Imagery**: High-resolution maritime/ROK Navy-themed images with CSS filters (Grayscale + Mix-blend Multiply Blue Tint).
- **Responsive Layout**: Compact, no-scroll dashboard style (1440px max-width).
- **Accessibility**: ARIA-compliant Dialog titles, keyboard-navigable components.

## 3. Page Structure
- `/`: Main Landing Page (Dashboard style)
- `/tips`: Shipboard Life Tips (함상 생활 가이드) - Informational resources for personnel.

## 4. Maintenance / AI Guidance
- **Design Rule**: Always maintain the "Modern Tactical Luxe" aesthetic. Avoid cluttered layouts.
- **Iconography**: Use `lucide-react`. Maintain consistency with Anchor, Waves, and Shield icons.
- **Localization**: Use formal, polished Korean terminology suitable for ROK Navy personnel.
