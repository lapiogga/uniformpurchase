This document defines the standard asset sources and generation strategies for the Navy Archive: Ocean Edition project.

## 1. UI/UX Mockups & Placeholders
- **Provider**: [Placeholder API](https://placehold.co/)
- **Usage**: Use for temporary image areas, profile placeholders, or structural wireframing.
- **Example**: `https://placehold.co/600x400/001a33/ffffff?text=Navy+Gear`

## 2. Illustrations & Vectors
- **Provider**: [unDraw](https://undraw.co/)
- **Usage**: Open-source SVG vectors for empty states, onboarding, or thematic storytelling.
- **Theme Color**: Use Navy Blue (`#001a33`) or Uniqlo Red (`#EE1C23`) as the primary accent for illustrations.

## 3. Iconography
- **Provider**: Lucide Icons CDN / Package
- **Usage**: Standard interface icons (Anchor, Waves, Shield, etc.).
- **Style**: Maintain consistent stroke width (usually 1.5 or 2) and sizing.

## 4. Visual Assets (Hero & Premium)
- **Primary Source**: AI Image Generation (Custom Assets)
- **Strategy**: Generate bespoke, high-end "Tactical Luxe" visuals using AI to ensure uniqueness and brand alignment.
- **Theme**: ROK Navy, Deep Blue, Digital Camouflage, Premium Apparel.
- **Post-processing**: Apply CSS filters (Grayscale + Deep Blue Multiply) to maintain the brand aesthetic.
